  • What does the meta tag do in HTML?

  • Defines metadata about the document.
  • Defines a paragraph of text.
  • Defines a link to an external resource.
  • Defines a section of the document.

  • Which HTML element is used to define the title of a document?

  • meta
  • head
  • title
  • header

  • What is the default behavior of the p tag in HTML?

  • It displays text in bold.
  • It creates a new paragraph with space above and below.
  • It centers the text.
  • It italicizes the text.

  • happens if you omit the closing tag in HTML?

  • The text will appear in uppercase.
  • The browser will automatically close the paragraph.
  • The paragraph will not be displayed.
  • The browser will throw an error.
